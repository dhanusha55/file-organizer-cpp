File Organizer (C++)

A CLI tool that automatically organizes files in a directory
into categories such as Images, Music, Documents, and Others.

Features:
- Automatic sorting by extension
- Summary counters
- Duplicate file detection
- Cross-platform filesystem support

How to Compile

bash
g++ file_organizer.cpp -std=c++17 -o organizer

 How to Run

./organizer <folder_path>


Example:

./organizer TestFolder
